# 02-express-restful-api

Express web server with a (mostly) RESTful API. To start:

```node
npm install
node server.js
```

## Endpoints

### `/tools`
Endpoint supports GET, POST, PUT and DELETE.

**Tools object**

Key      | Value
---------|------
name     | string
price    | number
quantity | number
